MyFlix Documentation

The purpose of this project is to sort information on movies, directors and genres of all the movies that you are interested in. You will be able to make a profile and save the movies that you've watched or want to watch and get information on them.

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies
Return info about a single movie by title to the user /movies/:title GET None A JSON object holding data about a single movie, containing description, genre and director.
Return data about a genre(description) by name/title. (e.g., "Thriller") /movies/genres/:genreName GET None A JSON object holding data about a genre and it's description
Return data about a director (name, bio, birth year) by name /movies/directors/:directorName GET None A JSON object holding data about a director, example: {name:"john", birth:"1997", death:"2000" bio:"..."
Allow a user to be created /users POST A JSON object holding data about the user to add, example: {Username: "john22", Password:"testing", Email:"john@gmail.com", Birthday:"02/03/1989"} A JSON object holding data about the user that was added, including an ID. Example: {"Username": "john22", "Password": "testing", "Email": "john@gmail.com", "Birthday": "1989-02-03T13:00:00.000Z", "FavoriteMovies": [], "_id": "6462145c67a9ae3c67f05a93",}
Allow users to update their user info (username) /users/:Username PUT Username and info they would like updated. A text indicating that the username was updated
Allow users to add a movie to their list of favourites /users/:Username/movies/:MovieID' POST A JSON object holding data about the user to add, example: {movieTitle: ""} A text indication that the movie has been added
Allow users to remove a movie from their list of favourites /users/:Username/movies/:MovieID DELETE Name of the movie which they want to removie example: {movieTitle:""} A text indication that the movie has been removed
Allow existing users to deregister /users/:Username DELETE None A text indication that the deletion was successful
